How to export for submission
Three ways to export at exact pixel dimensions
1
Chrome DevTools (easiest) Open the slide file → F12 → toggle device toolbar → type exact dimensions → right-click page → "Capture screenshot"
2
Puppeteer CLI (batch) Exports all slides at once with one command. Requires Node.js.
3
Browser zoom + GoFullPage Set browser zoom to match target width, install GoFullPage extension, capture.
# Puppeteer — install once, then export all slides
npm install -g puppeteer-screenshot-cli

puppeteer-screenshot --url ios-69-poster-1.html --width 1320 --height 2868 --output exports/slide-1.png
puppeteer-screenshot --url ios-69-poster-2.html --width 1320 --height 2868 --output exports/slide-2.png
puppeteer-screenshot --url ios-69-poster-3.html --width 1320 --height 2868 --output exports/slide-3.png
puppeteer-screenshot --url ios-69-strip.html --width 1320 --height 2868 --output exports/slide-4.png
puppeteer-screenshot --url android-feature.html --width 1024 --height 500 --output exports/feature-graphic.png
