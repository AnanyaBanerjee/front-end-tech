Features

A native client that speaks
the open protocol.

Live Streaming

Responses arrive token-by-token over SSE. You see the agent think in real time, not a loading spinner followed by a wall of text.

Multi-Agent Roster

Add unlimited agents. Organize them into groups and custom tabs. Switch between them like contacts in a messenger.

Connect by URL

No marketplace. Paste the endpoint, the app reads the Agent Card. The open web is the platform.

Rich Attachments

Send photos, images, and files inline. Multi-modal from day one.

Native Everywhere

iPhone, iPad, Mac, Vision Pro. One native binary per platform, not a web view.

Protocol

Three steps.
Zero ceremony.

If it publishes A2A metadata, AgentChat can talk to it. That is the entire integration surface.

01

Publish agent metadata

Host an Agent Card at your domain so clients discover capabilities automatically.

/.well-known/agent.json
02

Paste the URL into AgentChat

The app reads the Agent Card, discovers the endpoint, establishes a session. No SDK, no API key.

03

Start chatting

Stream responses, send files, manage conversations. Everything stored locally on your device.
