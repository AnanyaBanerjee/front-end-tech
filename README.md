# Front-End Tech

A collection of front-end skills, projects, and experiments.

## Skills

Design instruction files (from [taste-skill](https://github.com/Leonxlnx/taste-skill)) that guide AI tools to generate polished frontend UI:

| Skill | Style |
|-------|-------|
| **taste-skill** | Core design — layout, typography, colors, spacing, motion |
| **soft-skill** | Premium UI with whitespace and spring animations |
| **minimalist-skill** | Editorial, Linear/Notion-inspired interfaces |
| **brutalist-skill** | Swiss typographic and CRT terminal aesthetics |

Drop a `SKILL.md` into your project root and your AI coding tool will pick it up automatically.
